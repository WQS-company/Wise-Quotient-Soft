Design Prototypes
=================

These are lightweight, runnable prototypes demonstrating premium UI/UX for the top 3 features:

- `dashboard_widgets.html` — customizable widget dashboard (drag, resize, save layout).
- `analytics_benchmarks.html` — advanced analytics with interactive charts and benchmark toggle.
- `payments_escrow.html` — integrated payments & escrow dashboard mock.

How to run
----------
Open any of the HTML files in a browser (double-click or serve via local web server).

Notes
-----
- Prototypes use CDN-hosted Tailwind CSS and Chart.js for quick preview.
- Layout saves to `localStorage` so you can persist demo layouts in the browser.
- These are front-end prototypes for review and handoff; backend integrations are stubbed/mocked.
